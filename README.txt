# Backend Setup Instructions

1. Open a terminal and navigate to this directory:
   C:\Users\arnav\OneDrive\Documents\NEOMED\NepalTTS\Perplexity-Hackathon

2. Install the required Python packages:
   pip install -r requirements.txt

3. Start the backend server:
   python backend.py

4. Keep this terminal open while using the Tkinter app. The app will now be able to communicate with the backend at http://localhost:5000/triage.

The backend now uses hardcoded logic and does not require any AI or API keys.
